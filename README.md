## 2️⃣ 프로젝트 개요

- **여러 캠핑 전문 브랜드의 제품들을 한눈에 볼 수 있는 셀렉샵으로 최신의 캠핑용품을 소개해주고 원하는 상품을 구매할 수 있는 캠핑용품 판매 사이트**

<br>

## 3️⃣ 개발 배경 및 목표

- 개발 배경
  - **일반적인 캠핑 관련 브랜드에 웹사이트들은 자사 제품만 판매 하는 사이트입니다.**
  - **여러 브랜드의 제품을 판매하는 사이트라고 하더라도 블로그로 운영되거나 오프라인 매장 위치만 알려주는 웹사이트가 있습니다.**
  - **바쁜 시간으로 캠핑용품을 사러 갈 시간에 없거나 캠핑을 처음 접하는 소비자 관점에서 볼 때 상품을 하나의 웹사이트에서 구매가 이루어진다면 더욱 편리할 것으로 생각했습니다.**
  - **따라서 선호도가 높은 브랜드의 상품들을 한곳에 모아 사용자들이 다양하게 제품을 구매할 수 있도록 도와주는 웹사이트를 제작하고자 합니다.**
